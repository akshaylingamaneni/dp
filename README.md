# Screenshot Composer

Create beautiful screenshots with gradient backgrounds, customizable padding, shadows, and corner styling. open source.

## Features

- Gradient backgrounds and patterns
- Customizable padding, corner radius, and shadows
- Batch processing with consistent styling
- Corner text labels
- Export to PNG or ZIP

## Development

```bash
bun install
bun dev
```

Open [localhost:3000](http://localhost:3000)
